# Baseweight.com

A lightweight, offline-first hiking gear planner. Replaces LighterPack with a modern, darker, and more accessible interface.

## Features
- **Master Inventory**: Manage all your gear in one place.
- **Walk/Trip Planning**: Create specific packing lists from your inventory.
- **Visual Analytics**: Pie charts and weight breakdowns (Base weight, Consumable, Worn).
- **Dark/Light Mode**: Fully themeable with accessibility support.
- **Share & Export**: Download CSVs or share your list via a secure link.

## Hosting on Replit via GitHub

This project is ready to be hosted on Replit. Follow these steps:

### 1. Push to GitHub
1.  Create a new repository on [GitHub](https://github.com/new).
2.  Run the following commands in your terminal (if you haven't already):
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    git branch -M main
    git remote add origin https://github.com/YOUR_USERNAME/baseweight.com.git
    git push -u origin main
    ```

### 2. Import to Replit
1.  Log in to [Replit](https://replit.com).
2.  Click **"+ Create Repl"**.
3.  Choose **"Import from GitHub"**.
4.  Paste your new GitHub repository URL.
5.  Click **"Import from GitHub"**.

### 3. Run
Replit should automatically detect this as an HTML/CSS/JS project. Click **Run** to start the server.
